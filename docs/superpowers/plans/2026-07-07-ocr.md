# OCR(텍스트 추출) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 편집기에서 현재 이미지의 텍스트를 Vision OCR로 추출해 클립보드에 복사한다.

**Architecture:** `TextRecognizer`(Vision `VNRecognizeTextRequest` 래퍼)가 `CGImage`를 비동기 인식해 위→아래로 결합한 문자열을 반환한다. 편집기 툴바 버튼/단축키 `E`가 현재 `self.image`(crop 반영본, 주석 미포함 원본)를 인식해 `ClipboardWriter.write(text:)`로 복사하고 `Notifier`로 피드백한다. 결합 로직만 순수 함수로 분리해 단위 테스트.

**Tech Stack:** Vision, CoreGraphics, AppKit(NSPasteboard), XCTest.

- [ ] **Step 3: TextRecognizer 구현**

`Sources/SnapScreenKit/Editor/TextRecognizer.swift`:

```swift
import Vision
import CoreGraphics

/// Vision 온디바이스 OCR 래퍼. 이미지 → 인식 텍스트(비동기).
public enum TextRecognizer {
    /// 이미지를 OCR해 완료 핸들러로 결과 문자열을 돌려준다.
    /// 무거운 인식은 백그라운드에서 수행하고 콜백은 메인 액터에서 호출한다.
    public static func recognize(_ image: CGImage,
                                 completion: @escaping @MainActor (Result<String, Error>) -> Void) {
        let request = VNRecognizeTextRequest { request, error in
            if let error = error {
                Task { @MainActor in completion(.failure(error)) }
                return
            }
            let observations = (request.results as? [VNRecognizedTextObservation]) ?? []
            let lines = observations.compactMap { obs -> (text: String, minY: CGFloat)? in
                guard let top = obs.topCandidates(1).first else { return nil }
                return (top.string, obs.boundingBox.minY)
            }
            let text = joinedText(lines)
            Task { @MainActor in completion(.success(text)) }
        }
        request.recognitionLevel = .accurate
        request.recognitionLanguages = ["ko-KR", "en-US"]
        request.usesLanguageCorrection = true

        DispatchQueue.global(qos: .userInitiated).async {
            let handler = VNImageRequestHandler(cgImage: image, options: [:])
            do {
                try handler.perform([request])
            } catch {
                Task { @MainActor in completion(.failure(error)) }
            }
        }
    }

    /// (텍스트, 정규화 boundingBox minY) 목록을 위→아래(minY 내림차순)로 정렬해 줄바꿈 결합. 순수 함수.
    public static func joinedText(_ lines: [(text: String, minY: CGFloat)]) -> String {
        lines.sorted { $0.minY > $1.minY }.map(\.text).joined(separator: "\n")
    }
}
```

`performOCR()` 메서드 추가(`copyMerged` 근처, `import` 상단에 `import` 필요 시 — `TextRecognizer`/`ClipboardWriter`/`Notifier`는 같은 모듈이라 추가 import 불필요):

```swift
    @objc public func performOCR() {
        TextRecognizer.recognize(image) { [weak self] result in
            guard self != nil else { return }
            switch result {
            case .success(let text) where text.isEmpty:
                Notifier.show(title: "텍스트 없음", body: "이미지에서 인식된 텍스트가 없습니다")
            case .success(let text):
                ClipboardWriter.write(text: text)
                Notifier.show(title: "텍스트 복사됨", body: "\(text.count)자를 클립보드에 복사했습니다")
            case .failure(let error):
                Notifier.alertFailure(title: "OCR 실패", body: error.localizedDescription)
            }
        }
    }
```

(OCR 대상은 `image` — crop 반영된 원본. 주석 없는 이미지의 텍스트를 인식한다. `flattened()`가 아님에 주의.)
