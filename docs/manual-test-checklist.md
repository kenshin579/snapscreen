# 수동 테스트 체크리스트 (릴리스 전)

실기기(macOS)에서 릴리스 전 확인해야 하는 수동 검증 항목입니다. 자동화 테스트(`swift test`)로 커버되지 않는 UI/시스템 통합 동작을 다룹니다.

## 1. 권한 플로우

- [ ] 최초 실행 시 화면 기록 권한 안내 알림창이 표시된다
- [ ] "시스템 설정 열기" 클릭 시 시스템 설정 > 개인정보 보호 및 보안 > 화면 및 시스템 오디오 녹음 화면이 열린다
- [ ] 권한 허용 후 앱을 재시작하면 캡처가 정상 동작한다
- [ ] 최초 실행 시 알림(Notification) 권한 요청 다이얼로그가 표시된다
- [ ] 알림 권한을 거부한 상태에서 캡처 실패 시 비프음(beep)이 들린다

## 2. 전체 화면 캡처 (⌘⇧0)

- [ ] 캡처 후 편집기 창이 열린다
- [ ] 멀티 모니터 환경에서 마우스 포인터가 있는 화면이 캡처된다

## 3. 영역 캡처 (⌘⇧1)

- [ ] 오버레이가 어둡게 표시되고 십자선(crosshair)이 나타난다
- [ ] 드래그 중 선택 영역의 크기 라벨(px)이 표시된다
- [ ] 캡처 결과가 선택 영역과 픽셀 단위로 일치한다 (Retina 디스플레이에서 `sips`로 크기 확인)
- [ ] 첫 클릭 전 esc 키로 캡처가 취소된다
- [ ] 드래그 없이 클릭만 해도 캡처가 취소된다
- [ ] 보조 모니터에서도 선택 영역이 정확하게 캡처된다
- [ ] 단축키를 연타해도 오버레이가 1개만 생성된다

## 4. 창 캡처 (⌘⇧2)

- [ ] 마우스 아래에 있는 창이 하이라이트된다
- [ ] 겹쳐진 창 중 최전면 창이 우선적으로 선택된다
- [ ] 클릭으로 캡처하면 창 그림자 없이 내용만 캡처된다
- [ ] esc 키로 캡처가 취소된다
- [ ] 단축키를 연타해도 창 피커(picker)가 1개만 생성된다

## 5. 편집기 — 도구

- [ ] 화살표 도구로 그리기가 동작한다
- [ ] 사각형 도구로 그리기가 동작한다
- [ ] 원 도구로 그리기가 동작한다
- [ ] 모자이크 도구로 가린 영역의 글자를 읽을 수 없다
- [ ] 텍스트 입력: Enter로 커밋된다
- [ ] 텍스트 입력: esc로 취소된다 (주석이 생성되지 않음)
- [ ] 텍스트 입력: 한글 입력이 정상 동작한다
- [ ] 텍스트 커밋 시 위치/크기가 입력 중 화면과 비교해 점프하지 않는다
- [ ] 번호 배지 도구로 찍으면 ①②③ 순서로 번호가 자동 증가한다
- [ ] 도구 단축키(A/R/O/T/G/B/N)를 누르면 툴바 세그먼트 선택 상태와 동기화된다
- [ ] 블러 도구(G)로 가린 영역이 부드럽게 흐려진다 (모자이크와 달리 블록 패턴이 없음)
- [ ] 모자이크 도구 라벨/툴팁이 "모자이크"로 표시된다

## 6. 편집기 — 선택/편집

- [ ] 주석을 클릭하면 선택되어 점선 테두리가 표시된다
- [ ] 선택된 주석을 드래그하면 이동한다
- [ ] `⌘Z`로 이동 1회가 되돌려진다
- [ ] `⌘Z`를 연속으로 누르면 모든 동작이 취소된다
- [ ] `⇧⌘Z`로 취소된 동작이 다시 복원된다
- [ ] 선택된 주석에서 delete 키로 삭제된다
- [ ] 텍스트 입력 중 `⌘C`/`⌘Z`/`⌘S`를 눌러도 텍스트 필드 편집을 방해하지 않는다

## 7. 편집기 — 창

- [ ] 창 크기를 조절해도 이미지가 중앙에 aspect-fit으로 배치되고 레터박스가 생긴다
- [ ] 레터박스(이미지 바깥) 영역을 클릭해도 주석이 생성되지 않는다
- [ ] `⌘W`로 경고 없이 창이 닫힌다
- [ ] 연속 캡처 시 편집기 창이 여러 개 동시에 열린다
- [ ] 편집기 창을 반복해서 닫아도 크래시가 발생하지 않는다

## 8. 내보내기

- [ ] `⌘C`로 복사 후 다른 앱(예: 미리보기)에 붙여넣으면 주석이 포함된 이미지가 붙여넣어진다
- [ ] 툴바의 복사 버튼도 동일하게 동작한다
- [ ] `⌘S`로 저장하면 저장 후 편집기 창이 닫힌다
- [ ] 저장 파일명 형식이 `snapscreen YYYY-MM-DD HH.MM.SS.png`와 일치한다
- [ ] 같은 초에 2회 저장하면 두 번째 파일명에 " (2)"가 붙는다
- [ ] Retina 디스플레이에서 저장된 이미지의 픽셀 크기가 올바르다 (`sips`로 확인)
- [ ] PNG를 TIFF 형식으로만 받는 앱(예: 일부 구형 앱)에도 붙여넣기가 가능하다

## 9. 설정

- [ ] **다운로드한 릴리스 zip에서** 설정 창을 열어도 크래시하지 않는다 (KeyboardShortcuts 리소스 번들 누락 회귀 가드 — 로컬 빌드는 .build 때문에 통과하므로 반드시 배포 zip으로 확인)
- [ ] 설정 창이 열린다
- [ ] 단축키를 변경하면 즉시 반영된다
- [ ] 저장 폴더를 변경할 수 있고, "기본값" 버튼으로 되돌릴 수 있다
- [ ] 파일명 접두어를 변경하면 저장 파일명에 반영된다
- [ ] 앱을 재시작해도 설정이 유지된다
- [ ] 시스템 스크린샷 저장 위치(`defaults read com.apple.screencapture location`)를 변경하면 SnapScreen 저장 위치도 따라간다 (저장 폴더 오버라이드가 없을 때)

## 10. 일반

- [ ] 모든 창(홈·편집기·설정)을 닫은 상태에서는 Dock에 아이콘이 표시되지 않는다 (창이 열려 있을 때의 Dock 동작은 §12 참고)
- [ ] 메뉴바에 아이콘이 표시되고, 메뉴에 SnapScreen 홈… / 영역 캡처 / 창 캡처 / 전체 화면 캡처 / 설정… / SnapScreen 종료 6개 액션 항목(+구분선 3개)이 순서대로 표시된다
- [ ] 메뉴의 "SnapScreen 종료"로 앱이 정상 종료된다

## 11. 업데이트

- [ ] 설정 창 "정보" 섹션에 현재 버전이 표시된다
- [ ] "업데이트 확인" 클릭 시 최신 버전이면 "최신 버전입니다 ✓"가 표시된다
- [ ] 네트워크를 끊고 "업데이트 확인" 클릭 시 실패 메시지가 표시된다
- [ ] (구버전 설치 상태에서) 시작 시 자동 확인으로 메뉴바에 "업데이트 가능 (vX.Y.Z)…" 항목이 나타난다
- [ ] 메뉴바 업데이트 항목 클릭 시 설정 창이 열린다
- [ ] "업그레이드" 클릭 → 다운로드 → 앱이 재실행되고 새 버전으로 실행된다
- [ ] 업데이트 후 첫 실행에 "화면 기록 권한을 다시 켜야 할 수 있습니다" 알림이 표시된다 (알림 권한 허용 상태에서 확인할 것 — 거부 상태면 이 안내는 표시되지 않음)
- [ ] 업데이트 후 화면 기록 권한을 다시 켜면 캡처가 정상 동작한다

## 12. 홈 창

- [ ] 앱 실행 시 홈 창이 자동으로 뜨고 독 아이콘이 나타난다
- [ ] 홈 창의 캡처 버튼 3종(영역/창/전체)이 각각 정상 동작한다 (기존 캡처 흐름 시작)
- [ ] 홈 창의 버튼에 표시되는 단축키가 설정에서 변경한 값과 일치한다
- [ ] 홈 창을 닫으면 독 아이콘이 사라지고 메뉴바 상주만 유지된다
- [ ] 편집기 창이 여러 개 떠 있을 때 홈 창을 닫아도 독 아이콘이 유지된다 (창이 남아 있으므로)
- [ ] 모든 창(홈+편집기+설정)을 닫으면 독 아이콘이 사라진다
- [ ] 메뉴바 "SnapScreen 홈…"으로 닫은 홈 창을 다시 열 수 있다
- [ ] 모든 창을 닫아도 앱이 종료되지 않고 메뉴바에 상주한다
- [ ] 홈 창 표시 중 ⌘Tab에 SnapScreen이 노출된다 (.regular 정책)

## 13. 자르기 (Crop)

- [ ] 주석이 없을 때 툴바의 자르기 버튼이 활성화되어 있다
- [ ] 주석이 하나라도 있으면 자르기 버튼이 비활성(회색)이고 툴팁에 안내가 표시된다
- [ ] 자르기 버튼(또는 C 키)을 누르면 crop 모드로 들어가 화면이 어두워진다
- [ ] 드래그로 영역을 지정하면 그 부분만 밝아지고 테두리가 표시된다
- [ ] 드래그 후 선택 영역 근처에 확정(✓)/취소(✗) 버튼이 나타난다
- [ ] ✓ 또는 Enter로 확정하면 이미지가 그 영역으로 잘리고 창 크기가 새 비율에 맞게 바뀐다
- [ ] ✗ 또는 esc로 취소하면 이미지가 그대로 유지된다
- [ ] 자른 뒤 이어서 주석을 그릴 수 있다
- [ ] 자른 뒤 ⌘C/⌘S로 내보내면 잘린 이미지가 반영된다

## 14. 앱 아이콘

- [ ] Finder에서 SnapScreen.app이 캡처 스택(바이올렛) 아이콘으로 보인다
- [ ] Dock에 앱 창을 띄웠을 때 Dock 아이콘이 새 아이콘이다
- [ ] ⌘Tab 앱 전환기에서 새 아이콘이 보인다
- [ ] 시스템 설정 > 개인정보 보호 > 화면 기록 목록에서 새 아이콘이 보인다
- [ ] 작은 크기(목록/메뉴)와 큰 크기 모두 형태가 또렷하다

## 15. 펜 (자유곡선)

- [ ] 펜 도구(또는 P 키)를 선택하고 드래그하면 획이 그려진다
- [ ] 빠르게 그려도 곡선이 부드럽다(각지지 않음)
- [ ] 색 팔레트와 굵기가 획에 반영된다
- [ ] 그린 획을 클릭해 선택하고 드래그로 이동할 수 있다
- [ ] 선택한 획을 ⌫로, 또는 ⌘Z로 삭제/되돌릴 수 있다
- [ ] 클릭만 하고 그리지 않으면 빈 획이 생기지 않는다
- [ ] ⌘S/⌘C로 내보내면 그린 획이 반영된다

## 16. OCR (텍스트 추출)

- [ ] 한글 텍스트가 있는 이미지를 OCR(E 또는 버튼)하면 텍스트가 클립보드에 복사된다
- [ ] 영문 텍스트도 정상 추출된다
- [ ] 복사 후 알림에 글자 수가 표시된다
- [ ] 텍스트가 없는 이미지는 "텍스트 없음" 안내가 뜬다
- [ ] crop으로 자른 뒤 OCR하면 잘린 영역의 텍스트만 추출된다
- [ ] 추출한 텍스트를 다른 앱에 붙여넣으면 줄 순서가 위→아래로 맞다

## 17. 복사 피드백 토스트

- [ ] OCR로 텍스트를 복사하면 캔버스 하단에 "N자를 복사했습니다" 토스트가 뜨고 잠시 후 사라진다
- [ ] 텍스트 없는 이미지 OCR 시 "인식된 텍스트가 없습니다" 토스트가 뜬다
- [ ] ⌘C로 이미지를 복사하면 "이미지를 복사했습니다" 토스트가 뜬다
- [ ] 연속으로 복사하면 이전 토스트가 새 토스트로 교체된다
- [ ] OCR 실패 시에는 기존처럼 beep + 시스템 알림이 뜬다(토스트 아님)

## 18. 지우개

- [ ] 지우개(또는 X 키) 선택 시 마우스를 따라 원형 커서가 표시된다
- [ ] 펜 획 가운데를 문지르면 획이 두 조각으로 나뉜다
- [ ] 펜 획 끝을 문지르면 그 부분만 짧아진다
- [ ] 사각형·화살표·텍스트·번호 배지는 지우개가 닿으면 통째로 삭제된다
- [ ] 한 번 드래그로 여러 개를 지운 뒤 ⌘Z 한 번이면 전부 복원된다
- [ ] 지운 결과가 ⌘S/⌘C에 반영된다
- [ ] 창을 확대/축소해도 커서 원 크기와 실제 지워지는 범위가 일치한다
